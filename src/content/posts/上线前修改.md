# // TERMINAL UPDATE LOG: V1.2.0

随着全站上线进入倒计时，龙窝终端迎来了最后一次核心架构大修。本次更新主要聚焦于**桌面端交互美术的精细化打磨**以及**移动端全设备响应式的底层重构**，彻底解决了多端体验割裂以及内容截断的顽疾。

以下为本次系统更新的详细技术内幕：

---

## 🛠️ 1. 导航栏（Navbar）操控手感与视觉重构

为了强化“独立极客终端”的操控反馈，导航栏全面升级了交互算法：

* **磁吸按钮算法（Magnetic Hover）**：右侧二级菜单按钮引入了二维坐标向量计算。当鼠标划入按键引力场时，程序会精准捕获鼠标与按钮圆心的距离差（$\Delta X, \Delta Y$），并施加 `0.2` 的弹性引力系数，使按钮向鼠标方向平滑位移，产生极具物理质感的“磁吸吸附”效果。
* **斜向极客分割线**：取消了传统死板的竖直分栏，在按钮间嵌入了代码注释符号 `//`。利用数学剪切变换（`-skew-x-12`）让其向左倾斜 12 度，并大幅提升了在首屏全透明模式下的色彩亮度和像素厚度（`text-slate-400/70 font-bold`），确保在任何暗色背景下都锐利清晰。
* **组悬浮联动变色（Group Hover）**：左侧 Logo 区重构了组件层级。现在鼠标悬浮时，只有文字会同步平滑变换为琥珀色（`amber-400`），而网站 Icon 头像则保持原本的纯净与克制，不再产生冗余的外部辉光。
* **双路滚轮雷达系统**：针对主页“套娃舱布局”导致的浏览器原生 `window.scrollY` 失效问题，导航栏升级为双路监听机制。现在能够同时侦测全局窗口与内部全屏容器（`#fullpage-container`）的滚动高度，确保离开首屏 50 像素后，高级的深蓝磨砂玻璃质感（`backdrop-blur-md`）能瞬间无缝铺开。

## 🌌 2. 视觉画廊（Gallery）自动化引擎与下载拦截

对全站的图片资产管理进行了彻底的工程化改造，彻底废除了过去手动硬编码（Hardcoded）导入图片的落后生产力：

* **全自动扫描引擎**：利用 `import.meta.glob` 自动化技术，让系统自主扫描并加载 `src/assets/gallery/` 目录下的所有影像资产。以后更新照片，只需将文件拖入文件夹，站点便会自动渲染。
* **带高度保底的发牌器算法 v2.0**：为了在瀑布流无缝滚动中消除“相同照片紧挨在一起”以及“底部留白断流”的逻辑缺陷，算法引入了 `minItemsPerColumn` 铺满保底机制。不管图片池多小，都会通过取余运算将图片均匀、错落（`translate-y-8`）地分发给 4 条独立传送带，完美维持无缝循环的视觉幻觉。
* **Blob 数据流下载拦截器**：点击放大照片时激活琥珀色呼吸辉光（`animate-breath-glow`）与动态能量光尘粒子系统。针对保存本地功能，重构了底层网络请求，通过 `fetch` 拦截图片二进制数据流（Blob），强行剥离 Astro 在开发环境下的复杂长参数，并统一将后缀修正为轻量高画质的 `.webp` 格式，彻底解决了系统无法识别下载文件格式的 Bug。

## 📱 3. 移动端“太空舱套娃”布局与弹性解绑

针对手机竖屏生态下“内容空间被吃掉”、“查看全部按钮消失”等恶劣体验，进行了全站级别的响应式手术：

* **双轨制分流策略**：在 `index.astro` 层面实现响应式解绑。电脑端继续运行高精度的 JS 滚轮与鼠标拖拽物理切屏引擎；而手机端则全面让 JS 劫持逻辑“罢工放行”，完全交由浏览器原生的 CSS 滚动吸附（`snap-y snap-mandatory`）控场，规避了手势冲突。
* **太空舱嵌套滚动（Nested Scrolling）**：为所有内容区块穿上了独立高度的“外壳”（`h-[100dvh]`）与“内胆”。在手机上，长页面内部的内容会优先在舱内自行滚完，当滑至最底部时，下一次滑动才会触发外层卡槽，整屏“啪嗒”一下翻页，兼顾了翻页仪式感与内容完整性。
* **线下行程与社交网络弹性重组（FinalSection）**：彻底粉碎了死板的 `50vh` 屏幕高度锁死机制。在移动端，左侧的行程卡片（`Schedule Cards`）取消内部小滚动条，在主页面垂直完全铺开；右侧的社交矩阵则从电脑端的横向并排自动塌陷为垂直的单列/双列自适应网格（`grid-cols-1 sm:grid-cols-2`），配合页脚的 `mt-auto` 弹性撑开，彻底杜绝了排版挤压和乱套。
